# 🌅 Prompts ChatGPT - Recettes du Matin

## Instructions pour ChatGPT

Utilisez ces prompts exacts dans ChatGPT pour générer les images. Demandez à ChatGPT de créer des images en **800x600 pixels** avec un style cohérent.

---

## 1. Porridge aux fruits rouges

```
Créez une photographie culinaire professionnelle d'un bol de porridge d'avoine crémeux garni de fruits rouges frais (myrtilles, framboises, mûres), graines de chia saupoudrées, dans un bol blanc en céramique. Vue du dessus, éclairage naturel doux, fond blanc épuré, style moderne et appétissant. Format 800x600 pixels, haute qualité.
```

## 2. Smoothie bowl banane & épinards


## 🎯 Conseils pour ChatGPT

1. **Demandez à ChatGPT** : "Peux-tu créer ces 10 images avec un style cohérent ?"
2. **Précisez** : "Utilise le même style d'éclairage et de composition pour toutes"
3. **Format** : "Toutes les images doivent faire 800x600 pixels"
4. **Style** : "Style photographique professionnel, éclairage naturel, fond épuré"
