### **Format et Qualité**
- **Dimensions** : 800x600 pixels (ratio 4:3)
- **Format** : JPEG haute qualité
- **Style** : Photographie culinaire professionnelle
- **Cohérence** : Style uniforme pour toute la série

### **Composition Visuelle**
- **Angle de vue** : 
  - 70% vue du dessus (flat lay)

- **Éclairage** : Naturel, doux, lumineux
- **Fond** : Blanc épuré ou bois clair naturel
- **Mise en scène** : Épurée, ingrédients visibles, appétissante

## 🎨 Instructions de Style pour ChatGPT

### **Prompt Type Recommandé**
```
"Photographie culinaire professionnelle de [RECETTE], 
800x600px, éclairage naturel doux, vue du dessus/angle 45°, 
fond blanc épuré, ingrédients frais visibles, 
style moderne et appétissant, haute qualité"
```

### **Éléments Visuels Essentiels**
- **Couleurs** : Naturelles, vives mais harmonieuses
- **Textures** : Bien définies, appétissantes
- **Ingrédients** : Frais, colorés, reconnaissables
- **Ustensiles** : Modernes, épurés (bols blancs, cuillères dorées)
- **Mise en scène** : Quelques ingrédients autour du plat principal

## 🔄 Workflow de Production

### **Phase 1 : Test (5 images)**
1. Créer 1 image par moment de la journée
2. Valider le style et la cohérence
3. Ajuster si nécessaire

### **Phase 2 : Production (35 images restantes)**
1. Créer les images par lots de 10
2. Maintenir la cohérence stylistique
3. Vérifier la qualité de chaque image

### **Phase 3 : Intégration**
1. Télécharger toutes les images
2. Nommer selon la convention : `[moment]-[nom-recette].jpg`
3. Optimiser si nécessaire
4. Intégrer dans l'application
